# Pi-Pianoteq

Pi-Pianoteq is a Python/Midi remote control for Pianoteq

# Usage

First, check and update the configuration in `pi_pianoteq.conf`

Install dependencies  
`pipenv install`

Package the module  
`pipenv package`

Deploy to remote Raspberry Pi  
`pipenv deploy`

Or manual install  
`pip3 install pi_pianoteq-1.0.0`
